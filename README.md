# ⚡ Sentinel Mesh X  
### AI-Powered Disaster Intelligence System  

> **“From data to decisions — in real time.”**

---

## 🧠 Overview

Sentinel Mesh X is an AI-driven system that monitors environmental conditions, predicts disaster risk, and autonomously decides actions like **Monitor, Alert, or Evacuate**.

The system transforms raw environmental data into real-time decisions, helping reduce response time and potential damage during disasters.

---

## 🚀 Key Features

- 📡 Real-time environmental monitoring  
- 🤖 AI-based risk prediction  
- ⚡ Autonomous decision engine  
- 🚨 Emergency alert system  
- 📊 Interactive dashboard  
- 🌦 Scenario simulation (Calm / Warning / Storm)  
- 📜 Transparent decision logs  

---

## 🧠 System Architecture

- **Sensors:** Rainfall, wind, environmental inputs  
- **Prediction:** Risk score calculation  
- **Simulation:** Confidence modeling  
- **Decision:** AI selects action  
- **Action:** Monitor / Alert / Evacuate
